ColonyJS browser demo - create a team and make a payment

1) Let's connect to the Colony first Make sure you have the FUNDING and ADMINISTRATION permissions in the Root domain of that Colony

2) Next we create a team

3) We need to move some funds to the new team. Make sure the root colony has at least about 1 unit of the Colony's native token (see message below after connecting)

4) Then, within that team, send someone some of the Colony's native tokens
